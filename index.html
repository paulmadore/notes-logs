The which command in Linux is very useful for determining where things are located. There is no convention to this, not really, so it can be anywhere the user wants a given library to be. Python, for instance, is ordinarily in /usr/bin or /usr/local/bin. The fastest way to find out which of these is with the which command. For example, which python3 will tell where/if you have Python 3 installed.
